Project is on Enterprise Software Development.
Created a enterprise software "AfterLife" a funeral one-stop service that allows customers to choose the funeral packages they want without fuss.
The UI is connected to the microservices through HTTP REQUEST/RESPONSE with Flask and AMQP message exchange. 
Docker is used to containerize the microservices to adhere to a service of architecture style.
External APIs used are MailJET API, STRIPE SMS API, and TWILIO API.
